# decision-tree-random-forest

## 📌 Objective
To implement Decision Tree and Random Forest algorithms for classification and compare their performance.

## 🛠 Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📊 Dataset
The dataset contains:
- Age
- Sex
- Cholesterol
- Blood Pressure (BP)
- Target (0 = No Disease, 1 = Disease)

## ⚙️ Steps Performed
1. Loaded dataset using Pandas
2. Split data into training and testing sets
3. Trained Decision Tree model
4. Controlled overfitting using max_depth
5. Visualized Decision Tree
6. Trained Random Forest model
7. Compared accuracy of both models
8. Calculated feature importance
9. Applied cross-validation

## 📈 Results
- Decision Tree gives good accuracy
- Random Forest gives better accuracy and reduces overfitting

## 📚 Conclusion
Random Forest is more reliable than Decision Tree for this dataset.
